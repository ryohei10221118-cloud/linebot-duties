# 😴 懒人模式使用指南

这份指南专为使用「简化模式」的用户准备，超级简单，5 分钟搞定！

---

## 🎯 懒人模式是什么？

**懒人模式 = 只需要告诉 Bot 你的休息日，系统每天自动提醒你明天是否上班。**

你**不需要**：
- ❌ 维护复杂的班表
- ❌ 记录班别代码（N1/N2/M1 之类的）
- ❌ 设置组别
- ❌ 每天手动查询

你**只需要**：
- ✅ 第一次绑定身份（30 秒）
- ✅ 每月告诉 Bot 你的休息日（30 秒）
- ✅ 等待每天自动通知

---

## 📱 第一次使用（1 分钟）

### 步骤 1：加 Bot 为好友

用手机扫描管理员给你的 QR Code，加 Bot 为好友。

### 步骤 2：绑定身份

在 LINE 发送：
```
绑定 你的姓名
```

例如：
```
绑定 John
```

Bot 会回复：
```
✅ 绑定成功！

👤 姓名：John
📊 模式：简化模式

请设置你的休息日：
例如：休息日 11/3,11/10,11/17

设置后系统会每天自动提醒你！
```

### 步骤 3：设置本月休息日

查看你的班表，找出本月的休息日，然后发送：

```
休息日 11/3,11/10,11/17,11/24
```

**格式说明：**
- 用逗号 `,` 分隔
- 格式：`月/日`
- 例如：`11/3` 表示 11 月 3 日

Bot 会回复：
```
✅ 已设置休息日：

📅 11/3
📅 11/10
📅 11/17
📅 11/24

系统会在每天自动提醒你！
```

### ✅ 完成！

就这么简单！现在你会：
- **每天晚上 9 点**收到明天的通知
- **每天早上 9 点**收到今天的通知（如果你是夜班）

## 🔄 每月更新（30 秒）

当新的月份开始时，再次发送休息日即可：

```
休息日 12/1,12/8,12/15,12/22,12/29
```

系统会自动更新，不需要做其他操作。

## 💡 实用技巧

### 技巧 1：批量设置多个月

如果你想一次设置两个月的休息日：

```
休息日 11/3,11/10,11/17,11/24,12/1,12/8,12/15
```

系统会自动识别！

### 技巧 2：随时修改

如果休息日有变动，直接再发送一次新的即可：

```
休息日 11/3,11/10,11/18,11/24
```

（注意：这会**覆盖**之前的设置，不是追加）

### 技巧 3：快速查询

想知道明天上不上班？直接问：

```
明天上班吗
```

1 秒得到答案！

## ❓ 常见问题

### Q: 我需要每天告诉 Bot 我的班表吗？

**A:** 不需要！你只需要每月设置一次休息日，系统会自动提醒你。

### Q: 如果休息日临时改了怎么办？

**A:** 直接再发送一次新的休息日列表，系统会自动更新。

例如：
```
休息日 11/3,11/11,11/17,11/24
```
（原来 11/10 改成了 11/11）

### Q: 我可以看到别人的班表吗？

**A:** 不可以，简化模式只显示你自己的上班情况。

### Q: 如果我想用完整模式（看同班人员）怎么办？

**A:** 联系管理员，让他把你的班表加入系统，然后重新绑定：
```
绑定 你的姓名 组别
```
例如：
```
绑定 John M1组
```

### Q: 系统会在什么时候通知我？

**A:**
- 晚上 9 点：通知明天的情况
- 早上 9 点：通知今天的情况（针对夜班）

### Q: 我可以关闭通知吗？

**A:** 如果你想临时关闭通知，可以在 LINE 中：
1. 点击 Bot 聊天室右上角 → 设置
2. 关闭「通知」
3. 需要时再打开

（注意：这不会影响手动查询功能）

## ✨ 为什么选择懒人模式？

### 对比

| 功能 | 完整模式 | 懒人模式 |
|------|----------|----------|
| 设置复杂度 | ⭐⭐⭐ | ⭐ |
| 每月维护时间 | 10-30 分钟 | 30 秒 |
| 查看同班人员 | ✅ | ❌ |
| 查看班别代码 | ✅ | ❌ |
| 自动通知 | ✅ | ✅ |
| 手动查询 | ✅ | ✅ |

### 适合人群

**懒人模式适合你，如果：**
- ✅ 你只关心明天是否上班
- ✅ 你不需要知道是早班还是夜班
- ✅ 你不需要查询同班人员
- ✅ 你想用最少的时间管理班表

**完整模式适合你，如果：**
- ✅ 你需要知道具体班别（N1/M2/A1）
- ✅ 你需要查询同班人员
- ✅ 你的班表很复杂
- ✅ 你不介意多花一点时间维护

## 🎓 进阶使用（可选）

### 设置提醒文字

如果你想在休息日时看到特定的提醒，可以：

1. 在 LINE 的「记事本」中记录每个休息日的计划
2. 或者使用 LINE 的「Keep」功能

### 结合行事历

你可以：
1. 把休息日加入手机行事历
2. 设置提醒
3. Bot 通知 + 行事历 = 双重保险

## 🆘 需要帮助？

如果遇到问题：

1. **首先**，发送「帮助」查看命令
2. **然后**，检查是否已经绑定身份
3. **最后**，联系管理员

常见错误：
- 「❌ 请先绑定身份！」→ 发送 `绑定 你的姓名`
- 「❌ 你还没有设置休息日」→ 发送 `休息日 11/3,11/10`
